
	I. 관리자 : 게시판 생성시 설정 메뉴
		a. DisPlay Type : 리스팅 형태 - 기본
			i. News : 리스트 -기본
			ii. Photo : 카드형? -기본
			iii. WebZine : 좌측 이미지 띄우는 리스트
		b. User Defined : 리스팅 형태 사용자가 변경 가능
		c. Title(필수) : 게시판 명
		d. Initial Page : 게시판 초기화면
			i. View : 리스트
			ii. View Message : 최신글 상세보기
		e. Color Scheme : 색상테마?(타이틀 색상)
			i. Gray
			ii. Blue
			iii. Red

	II. 게시판 작성시 설정 메뉴
		a. Sticky Enabled : 날짜지정?(종료시 없어지는듯?) - 제외
		b. Secret Enabled : 비공개 – 사용자가 게시글 작성시 선택가능
		c. Anonymous Enbled : 익명으로 게시 - 제외
		d. Editor Enbled : 게시글 서식 에디터 추가 – 기본적으로 적용 되게
		e. File Count(필수) : 업로드 가능 파일갯수 지정 – 기준 정해서 기본 적용
		f. File size(Byte) : 업로드 가능 용량 지정 – 기준 정해서 기본 적용
		g. File Extension : 업로드 가능 확장자 지정 – 기준 정해서 기본 적용
		h. Message No(필수) :리스팅갯수 – 기준 정해서 기본적용
		i. Hot Icon : 인기글 표시(몇클릭이상) - 기준 정해서 기본적용
		j. New Icon (필수) (몇일 유지) - 기준 정해서 기본 적용
		k. Message Emailed By - 제외
		l. Email Title(Message)
			i. You can use following mail-tags:
			[$title$], [$writer$], [$date$]
		m. Email Contents(Message)
			i. You can use following mail-tags:
			[$title$], [$writer$], [$content$], [$date$], [$link$]
		n. Comment Enabled : 댓글 활성화 – 사용자가 게시글 작성시 선택가능
		e. Comment Emailed : 댓글이 추가될시 메일을 받을것인지?
                                                            - 게시글 작성시 선택가능





	III. 코멘트 작성시 설정 메뉴
		a. Comment Editor Enabled : 댓글 서식 에디터 추가 - 제외
		b. Comment File count (Required) : 댓글 파일첨부 - 제외
		c. Comment File Size (Required) : 댓글 파일첨부 용량 - 제외
		d. Comment File Extension : 댓글 파일첨부 가능 확장자 - 제외
		e. Email Title (Comment)
			i. You can use following mail-tags: [$writer$], [$date$]
							- 기준 정해서 기본 적용
		f. Email Contents(Comment)
			i. You can use following mail-tags:
			[$writer$], [$content$], [$date$], [$link$]
							- 기준 정해서 기본 적용
	IV. 리스트상 표시 항목
		a. Show no in View : 목록 번호 보이기 - 기본
		b. Show Category in view : 말머리 보이기 - 제외
		c. Show File in View : 파일 보이기
		d. Show Comment Count in view : 댓글 수 보이기 -기본
		e. Show Writer in View : 작성자 보이기 -기본
		f. Show Date in View : 작성(수정)날짜 보이기 - 기본
		g. Show View Count in View : 접근수 보이기 - 기본
		h. Show Content in view : 글내용 보이기 - 제외
	V. 게시글 상세보기 항목
		a. Show Category in View Message : 상세보기 말머리 보이기 - 제외
		b. Show File in View Message : 파일 보이기 -기본
		c. Show Comment in View Message : 댓글 보이기 -기본
		d. Show Writer in View Message : 작성자 보이기 -기본
		e. Show Date in View Message : 날짜 보이기 - 기본
		f. Show View Count in View Message : 접근 수 보이기 - 기본
	VI. Comment 항목
		a. Show File in Comment : 파일 보이기 - 제외
		b. Show Date in Comment : 날짜 보이기
